# 🧱 CTA Component – User Stories

***

# 🔷 1. Core CTA Story (primary conversion)

### User story

**As a potential client nearing the end of the page**,  
I want a clear and simple next step,  
so that I can engage without friction or pressure.

### Acceptance criteria

* The CTA section is visually distinct from surrounding content
* The message clearly invites action without urgency
* Only **one primary action** is presented
* The button is clearly visible above the fold of the CTA section
* The user understands what will happen next (e.g. “conversation”, not “submit form”)

***

# 🔷 2. Headline story (tone setting)

### User story

**As a business decision-maker**,  
I want the CTA message to feel aligned with the calm tone of the site,  
so that I don’t feel pressured or marketed to.

### Acceptance criteria

* Headline uses simple, human language:
  * e.g. “Ready to bring clarity to your workplace?”
* No urgency phrases (e.g. “Act now”, “Don’t miss out”)
* No exaggerated claims
* The tone reflects confidence and restraint

***

# 🔷 3. Supporting copy story

### User story

**As a cautious buyer**,  
I want reassurance before taking action,  
so that I feel comfortable initiating contact.

### Acceptance criteria

* Supporting text reduces friction:
  * e.g. “Let’s have a conversation”
* Copy implies a low-commitment interaction
* No forms, pricing, or commitments introduced at this stage
* Language feels open and approachable

***

# 🔷 4. Primary button story

### User story

**As a user ready to engage**,  
I want a clear and easy action to take,  
so that I can proceed without confusion.

### Acceptance criteria

* Button label is clear and aligned with tone:
  * ✅ “Start a conversation”
  * ❌ “Submit”, “Register”, “Buy”
* Button is visually prominent using brand accent colour
* Hover state provides subtle feedback (darken or lift)
* Button is accessible (contrast, focus state)

***

# 🔷 5. Interaction / behaviour story

### User story

**As a user navigating the site**,  
I want a smooth and predictable transition when I click the CTA,  
so that the experience feels seamless.

### Acceptance criteria

* CTA links to a clearly defined next step:
  * contact page OR modal OR booking page
* Interaction is immediate (no delays or confusion)
* If modal is used:
  * appears smoothly
  * easy to close
* No unexpected redirects

***

# 🔷 6. Visual hierarchy story

### User story

**As a scanning user**,  
I want to immediately recognise this section as the next step,  
so that I don’t miss the opportunity to engage.

### Acceptance criteria

* Section uses subtle background differentiation:
  * e.g. `--accent-soft` or light tint
* Headline is larger than surrounding sections
* Button is the most visually prominent element
* Spacing around elements is generous

***

# 🔷 7. Emotional transition story

### User story

**As a visitor who has just understood the problem and solution**,  
I want to feel confident taking the next step,  
so that I move forward naturally.

### Acceptance criteria

* CTA follows logically after:
  * outcome
  * services
  * trust content
* The user feels:
  * reassured
  * understood
  * not rushed
* The CTA reinforces confidence, not urgency

***

# 🔷 8. Accessibility & clarity story

### User story

**As a user with varying needs**,  
I want the CTA to be clear and accessible,  
so that I can interact with it easily.

### Acceptance criteria

* Button meets contrast requirements
* Text is readable at all screen sizes
* Focus state is visible (keyboard navigation)
* Click/tap target is large enough (mobile-friendly)

***

# 🔷 9. Responsive behaviour story

### User story

**As a mobile user**,  
I want the CTA to remain clear and usable,  
so that I can take action easily on any device.

### Acceptance criteria

* Text stacks cleanly on smaller screens
* Button spans full width (or prominent width) on mobile
* Spacing remains comfortable (no cramped layout)
* No overlap or clipping of text

***

# 🔷 10. Brand alignment story (important)

### User story

**As a potential client evaluating trust**,  
I want the CTA to reflect professionalism and restraint,  
so that I feel confident engaging with the business.

### Acceptance criteria

* No aggressive colours or animations
* Motion (if any) is subtle (e.g. hover lift)
* Language aligns with consulting tone
* Section feels calm, not high-pressure

***

# 🔷 11. Optional secondary path (future enhancement)

*(You may not use this initially, but worth defining now)*

### User story

**As a user not ready to engage directly**,  
I want an alternative next step,  
so that I can continue exploring at my own pace.

### Acceptance criteria

* Optional secondary link:
  * “View our work”
  * “Learn how it works”
* Secondary action is visually less prominent
* Does not compete with primary CTA
