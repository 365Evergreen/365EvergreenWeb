# Separator Block

Story: A visual divider used to separate sections of content with an optional style (dots, solid, wide) and alignment.

Example:

```
<!-- Separator between two paragraphs -->
```

Acceptance Criteria:
- Renders a visible divider matching chosen style and alignment.
- Supports setting thickness, color, and width.
- Does not break semantics; still allows screen readers to navigate content.
- Spacing above and below can be adjusted and persists.
- Responds to theme color tokens for automatic contrast.
