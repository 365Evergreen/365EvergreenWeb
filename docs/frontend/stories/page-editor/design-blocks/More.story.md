# More Block

Story: A simple block used to insert a "More" tag that splits content for excerpts and read-more links on archive pages.

Example:

```
<!-- More tag inserted between paragraphs -->
```

Acceptance Criteria:
- Inserting the block creates an excerpt break when rendering lists/archives.
- The editor shows a clear divider where the break occurs.
- Read-more link is generated on archive pages and navigates to the full post.
- Block is non-visual in final output aside from the excerpt behavior.
- Works across themes that support excerpts and more-links.
