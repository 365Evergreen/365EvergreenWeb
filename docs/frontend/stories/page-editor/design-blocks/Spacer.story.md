# Spacer Block

Story: An invisible spacing block that adds adjustable vertical space between blocks to fine-tune layout rhythm.

Example:

```
<!-- Spacer set to 48px between sections -->
```

Acceptance Criteria:
- Spacer height is configurable and reflects exact spacing in the editor.
- Spacer collapses to zero height when set to 0.
- Spacer is non-interactive in published output (purely layout).
- Editor shows handles or controls for precise adjustments.
- Spacing remains consistent across responsive breakpoints.
