# Stack Block

Story: A vertical layout block that stacks child blocks with configurable spacing and alignment. Use for vertical groupings where order matters.

Example:

```
<!-- Stack: headline, paragraph, button -->
```

Acceptance Criteria:
- Blocks are stacked vertically with consistent spacing.
- Spacing, alignment, and reverse-order options available.
- Supports nested blocks and preserves their settings.
- Responsive behavior maintains readable spacing on small screens.
- Maintains accessible reading order for assistive tech.
