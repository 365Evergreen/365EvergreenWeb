# Row Block

Story: A horizontal layout block used to place blocks in a single row with alignment and distribution controls. Ideal for toolbars or side-by-side content.

Example:

```
<!-- Row with three items spaced evenly -->
```

Acceptance Criteria:
- Items align horizontally and respect distribution settings (left, center, space-between).
- Items wrap or overflow behavior is configurable.
- Row settings apply consistently across responsive breakpoints.
- Items remain independently editable.
- Keyboard navigation between items is intuitive.
