# Grid Block

Story: A two-dimensional layout block that arranges child blocks into rows and columns with explicit grid controls. Use for galleries, feature grids, and card layouts.

Example:

```
<!-- Grid: 3 columns, auto-flow dense -->
```

Acceptance Criteria:
- Grid allows configuring columns, rows, and gap sizes.
- Child items can span multiple columns/rows.
- Grid is responsive and reflows to fit viewport width.
- Items maintain focus and keyboard navigation order.
- Visual layout matches spacing tokens and design guidelines.
