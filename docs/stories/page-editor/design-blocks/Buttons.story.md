# Buttons Block

Story: A container for one or more call-to-action buttons. Use to group related actions like primary/secondary actions on a page.

Example:

```
<!-- Buttons: Primary and Secondary -->
```

Acceptance Criteria:
- Buttons render in a horizontal row by default and wrap responsively.
- Support primary, secondary, and link/button styles.
- Proper keyboard focus and visible focus ring.
- Accessible labels and roles present; icons have alt text or aria-hidden.
- Spacing and sizing follow design tokens.
