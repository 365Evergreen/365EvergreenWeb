Create your own block

Can’t find a block that does what you need? You can create your own using Telex, a free experimental AI tool from Automattic.

Describe the block you want in plain language, and Telex generates a working block you can download and install on your site. For example, you could create:

  • A recipe ingredient list with adjustable servings
  • A scratch-off reveal for hidden content
  • A personality quiz with shareable results
  • A reading progress indicator

Visit the Telex FAQ to learn more. Creating a block with Telex is completely free. You will install the block on your site the same way you would install a plugin, which requires a site with a Personal plan or higher.
